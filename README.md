# RTasks

RTasks is a simple Rust CLI Tool to manage tasks.

WIP


## Usage

```bash
./RTasks example.csv
```
RTasks saves to a csv file, which holds all the tasks. When an empty or non-existent csv is supplied, a new one of the name will be created.
